# Activating Basic Authentication Scheme in Java

Since Java 8u111 (opens new window), the Basic authentication scheme has been deactivated, by default, in the Oracle Java Runtime, by adding Basic to the jdk.http.auth.tunneling.disabledSchemes networking property. If your proxy requires it, the Basic authentication scheme can be reactivated by removing Basic from the jdk.http.auth.tunneling.disabledSchemes networking property:

jdk.http.auth.tunneling.disabledSchemes=

In Java 8, this property may be configured in $JAVA_HOME/jre/lib/net.properties. From Java 9 onwards, the path changed to $JAVA_HOME/conf/net.properties.
